# Showcase-Red-App
## Overview
The ShowCase Red App has been created to give developers, building systems using Sabre Red Developer Toolkit, an overview of the SDK capabilities and integration points between the End User (Travel Agent), Sabre Red 360 POS Application and DevStudio APIs.
